fun <T : Any> argPartition(a: KtNDArray<T>, kth: Int, axis: Int = -1, kind: String = "introselect"): KtNDArray<Long> (source)

Perform an indirect partition along the given axis using the algorithm specified by the kind keyword.

Parameters

a - array to sort.

kth - element index to partition by.

axis - along which to sort. Th default is the last axis (-1).

kind - selection algorithm. Default is introselect.

Return
KtNDArray of indices that partition a along the specified axis.
