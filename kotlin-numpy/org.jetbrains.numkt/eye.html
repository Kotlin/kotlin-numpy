fun <reified T : Any> eye(n: Int, m: Int? = null, k: Int = 0, order: Order = Order.C): KtNDArray<T> (source)

Return a 2-D array with ones on the diagonal and zeros elsewhere.

Parameters

n - number of rows in the output.

m - number of columns in the output. If null, defaults to n.

k - index of the diagonal.

order - see Order

Return
KtNDArray where all elements are equal to zero, except for the k-th diagonal, whose values are equal to one.
