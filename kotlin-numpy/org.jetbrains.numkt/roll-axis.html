rollAxis

fun <T : Any> rollAxis(a: KtNDArray<T>, axis: Int, start: Int = 0): KtNDArray<T> (source)

Roll the specified axis backwards, until it lies in a given position.

Parameters

a -

  • input array KtNDArray of type T

axis -

  • the axis to roll backwards. The positions of the other axes do not change relative to one another.

start -

  • the axis is rolled until it lies before this position. The default - 0.

Return
view of the input array.
