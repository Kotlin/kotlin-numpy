repeat

fun <T : Any> repeat(a: KtNDArray<T>, reps: Int, axis: Int? = null): KtNDArray<T> (source)

Repeat elements of an array.

Parameters

a - input array.

reps - the number of repetitions for each element. repeats is broadcasted to fit the shape of the given axis.

axis - the axis along which to repeat values. By default, use the flattened input array, and return a flat output array.

Return
output array which has the same shape as a, except along the given axis.
