copyto

fun <R : Any, T : Any> copyto(dst: KtNDArray<R>, src: KtNDArray<T>, casting: Casting = Casting.SAME_KIND): Unit (source)

Copies values from one array to another, broadcasting as necessary.

Two arrays must be of the same type and match in size, otherwise raise ValueError.

Parameters

dst - the array into which values are copied.

src - the array from which values are copied.

casting - see Casting.
