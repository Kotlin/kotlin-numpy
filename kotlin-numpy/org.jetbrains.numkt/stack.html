fun <T : Any> stack(vararg arrs: KtNDArray<T>, axis: Int = 0): KtNDArray<T> (source)

Join a sequence of arrays along a new axis.

Parameters

arrs - the input arrays. Each array must have the same shape.

axis - the axis in the result array along which the input arrays are stacked. Default is 0.

Return
new KtNDArray. The stacked array has one more dimension than the input arrays.
