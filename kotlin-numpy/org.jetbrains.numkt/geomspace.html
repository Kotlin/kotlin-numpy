fun <reified T : Number> geomspace(start: Number, stop: Number, num: Int = 50, endpoint: Boolean = true, axis: Int = 0): KtNDArray<T> (source)

Return numbers spaced evenly on a geometric progression.

Parameters

start - the starting value of the sequence.

stop - the final value of the sequence.

num - number of samples to generate. Default is 50.

endpoint - if true, stop is the las sample.

axis - in the result to store the samples.

Return
num samples, equally spaced on a log scale.
