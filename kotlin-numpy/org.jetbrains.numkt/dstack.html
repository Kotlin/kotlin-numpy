fun <T : Any> dstack(vararg tup: KtNDArray<T>): KtNDArray<T> (source)

Stack arrays in sequence depth wise (along third axis).

Parameters

tup - the arrays must have the same shape along all but the third axis. 1-D or 2-D arrays must have the same shape.

Return
The array formed by stacking the given arrays, will be at least 3-D.
