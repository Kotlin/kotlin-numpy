unpackbits

fun unpackbits(a: KtNDArray<Byte>, axis: Int? = null, count: Int? = null, bitorder: String = "big"): KtNDArray<Byte> (source)

Unpacks elements of a uint8 array into a binary-valued output array.

Parameters

a - input array.

axis - the dimension over which bit-unpacking is done.

count - the number of elements to unpack along axis, provided as a way of undoing the effect of packing a size that is not a multiple of eight.

bitorder - the order of the returned bits.

Return
the elements are binary-valued (0, or 1).
