fun <T : Any> vstack(vararg tup: KtNDArray<T>): KtNDArray<T> (source)

Stack arrays in sequence vertically (row wise).

Parameters

tup - the arrays must have the same shape along all but the first axis. 1-D arrays must have the same length.

Return
The array formed by stacking the given arrays, will be at least 2-D.
