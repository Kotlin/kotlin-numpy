partition

fun <T : Any> KtNDArray<T>.partition(kth: IntArray, axis: Int = -1, kind: String = "introselect"): Unit (source)

Rearranges the elements in the array in such a way that the value of the element in kth position is in the position it would be in a sorted array.

Parameters

kth - element index to partition by.

axis - axis along which to sort. Default is -1, which means sort along the last axis.

kind - selection algorithm. Default is 'introselect'.
