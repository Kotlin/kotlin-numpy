NPY_ITER_DONT_NEGATE_STRIDES

NPY_ITER_DONT_NEGATE_STRIDES (source)

This only affects the iterator when NPY_KEEPORDER is specified for the order parameter. By default with NPY_KEEPORDER, the iterator reverses axes which have negative strides, so that memory is traversed in a forward direction. This disables this step.
